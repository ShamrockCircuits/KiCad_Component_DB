# Taxonomy
All components share the same base format `CCC-NNN-VVVV`. To help organize this component category, 
mechanical connectors have used the `NNN` digits to classify the mechanical family as described below.

- `N`xx - Mechanical Family
    - `0`xx - <u>Unassigned</u>
    - `1`xx - Generic (Board-to-board / Wire-to-board)
        - `10`x - <u>Unassigned</u>
        - `11`x - Socket, Plug
        - `12`x - Header, Pin strip
        - `13`x - Terminal Block (Screw)

    - `2`xx - Generic Wire-to-board
        - `20`x - <u>Unassigned</u>

    - `3`xx - RF (SMA, MMCX, BNC)
        - `30`x - <u>Unassigned</u>

    - `4`xx - Modular I/O Connectors (RJxx, DBxx, ect)
        - `40`x - <u>Unassigned</u>

    - `5`xx - USB (all types)
        - `50`x - <u>Unassigned</u>

    - `6`xx - Edge Connectors
        - `60`x - <u>Unassigned</u>

    - `7`xx - Power Connectors (Barrel Jack, ATX, ect)
        - `70`x - <u>Unassigned</u>

    - `8`xx - <u>Unassigned</u>
    - `9`xx - Other

- xx`N` - Specifies the number of pins
    - xx`0` - More than 8
    - xx`1` - 1 Pin
    - xx`2` - 2 Pin
    - xx`3` - 3 Pin
    - xx`4` - 4 Pin
    - xx`5` - 5 Pin
    - xx`6` - 6 Pin
    - xx`7` - 7 Pin
    - xx`8` - 8 Pin
    - xx`9` - Other
